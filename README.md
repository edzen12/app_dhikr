# flutter_first_app

A new Flutter project.

#### Виджеты во FLutter - это "кирпичик", и как "матрешка"
#### Функция runApp -MaterialApp > Scaffold
#### - MaterialApp - обертка всего приложения
#### - Scaffold - обертка одного экрана
#### Column - содержит виджеты сверху вниз
#### Row - содержит виджеты слева на право
#### Text - просто вывести текст
#### Container - блок, который прорисует любую область, многофункциональный блок
#### Padding - виджет задающий внутренние отступы
#### Expanded - виджет,который растянет дочерний виджет на всё доступное место
